How It Works

1

Type or Select Text

Type in any input field or select text anywhere on a page. The ✨ button appears automatically.

2

Choose a Coach

Pick from English, Translator, Clarity, Tone, or any other enabled coach. Change the model per coach.

3

Get Coaching

See corrections, suggestions, and tips in scrollable sections. Apply the corrected text back with one click.
